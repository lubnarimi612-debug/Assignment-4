1. Difference between getElementById, getElementsByClassName, and querySelector?

getElementById: শুধু আইডি দিয়ে একটি নির্দিষ্ট এলিমেন্ট খুঁজে বের করে।

getElementsByClassName: একই ক্লাসের সব এলিমেন্টকে একটি লিস্ট হিসেবে দেয়।

querySelector: সিএসএস স্টাইলে আইডি, ক্লাস বা ট্যাগ—সবকিছু দিয়েই এলিমেন্ট ধরা যায়। এটি শুধু প্রথম এলিমেন্টটি দেয়, আর querySelectorAll দিলে সবগুলোকে লিস্ট হিসেবে দেয়।

2. How to create and insert a new element into the DOM?
প্রথমে document.createElement() ব্যবহার করে নতুন ট্যাগ বানাতে হয়। তারপর appendChild() বা append() মেথড দিয়ে সেটিকে এইচটিএমএল-এর কোনো নির্দিষ্ট জায়গায় বা প্যারেন্টের ভেতরে বসিয়ে দিতে হয়।

3. What is Event Bubbling?
ইভেন্ট বাবলিং হলো কোনো চাইল্ড এলিমেন্টে ক্লিক করলে সেই ইভেন্টটা বুদবুদের মতো উপরে উঠতে থাকে এবং তার প্যারেন্ট এলিমেন্টগুলোকেও ট্রিগার করে। অর্থাৎ অ্যাকশনটা নিচ থেকে উপরে ট্রান্সফার হয়।

4. What is Event Delegation and why is it useful?
আলাদা আলাদা অনেকগুলো চাইল্ড এলিমেন্টে ইভেন্ট লিসেনার না বসিয়ে, তাদের কমন প্যারেন্টে একটি মাত্র লিসেনার বসানোকে ডেলিগেশন বলে। এটি কোড কমায় এবং মেমোরি সাশ্রয় করে, বিশেষ করে যখন ডাইনামিকভাবে নতুন এলিমেন্ট যোগ হয় তখন এটি খুব কাজে লাগে।

5. preventDefault() vs stopPropagation()?

preventDefault(): এটি ব্রাউজারের কোনো ডিফল্ট কাজ আটকায়।

stopPropagation(): এটি ইভেন্ট বাবলিং থামায়, যাতে ইভেন্টটি তার উপরের প্যারেন্টদের কাছে পৌঁছাতে না পারে।
